# Maven Repository
### How to use
```
<repositories>
  <repository>
    <id>git-ebikyatto</id>
    <name>ebikyatto's maven repository</name>
    <url>https://github.com/ebikyatto/maven-repo/raw/master/</url>
  </repository>
</repositories>
```
### Dependencies
- **Microsoft SQLServer JDBC 4.1**
```
<dependency>
  <groupId>com.microsoft.sqlserver</groupId>
  <artifactId>sqljdbc</artifactId>
  <version>4.1</version>
</dependency>
```
